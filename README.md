# El impacto de las nuevas tecnologías en la sociedad: visualización del futuro

## Generador Seguro de Contraseñas

Actualmente, con el avance de la tecnología, han surgido muchos beneficios, pero también muchas amenazas, especialmente en la parte de la seguridad digital. Uno de los problemas muy comunes suele ser el hecho de que las contraseñas que usamos a diario en diferentes cuentas personales y sitios web no suelen ser muy seguras, por lo que es muy fácil lograr recibir algún ataque digital, como robos de identidad, accesos no autorizados o incluso vulneración de información personal.

Este programa responde a esta problemática proporcionando una herramienta práctica que genera contraseñas seguras mediante las opciones que desee el usuario usar, además de que evalua cada contraseña indicandole al usuario si la contraseña que se ha generado con respecto a sus opciones elegidas es segura o si de lo contrario, es débil.

# Propósito del Proyecto
El proposito de este proyecto es analizar el impacto que se ha generado por el creciente desarrollo de nuevas tecnologías enfocandonos en crear una herramienta útil que pueda ayudar a los usuarios y que esté correlacionado con la ciberseguridad.

# Objetivos del Proyecto
Crear un generador seguro de contraseñas que permita al usuario crear contraseñas seguras con opciones como:
- Letras mayúsculas
- Letras minúsculas
- Números
- Símbolos especiales
- Longitud personalizada

# Funcionalidades del Generador Seguro de Contraseñas

## 1. Ingreso de longitud de la contraseña
El usuario deberá ingresar los caracteres que desea usa en la contraseña.

## 2. Validación de entrada
El sistema verifica si la entrada del usuario contiene letras, si lo hace, mostrará un mensaje indicando que solo puede ingresar números, no letras.

## 3. Selección de tipos de caracteres
El usuario puede elegir entre:
* Mayúsculas
* Minúsculas
* Símbolos

## 4. Validación de selección
El sistema verifica si el usuario seleccionó al menos una opción, si no lo hizo, mostrará un mensaje indicando que debe seleccionar al menos una opción.

## 5. Construcción del conjunto de caracteres
El programa crea una cadena con los tipos de caracteres seleccionados por el usuario.
